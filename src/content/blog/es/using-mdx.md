MDX es útil cuando un post normal en markdown necesita un pequeño bloque de UI interactivo o reutilizable.

## Por qué lo uso para notas de Linux

Mantengo la mayoría de los posts en markdown plano, pero MDX es genial para listas de verificación repetidas.

<ul>
	<li>Confirmar que los backups están en buen estado.</li>
	<li>Ejecutar actualizaciones de paquetes con las notas de versión abiertas.</li>
	<li>Reiniciar solo después de comprobar los servicios en ejecución.</li>
</ul>

## Mantén MDX aburrido

Usa MDX para pequeños fragmentos reutilizables, no para lógica completa de aplicación. Cuanto más limpios permanezcan tus archivos de contenido, más fácil será mantenerlos.
