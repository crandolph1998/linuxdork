Start with built-in tools before installing anything:

1. `top` or `htop` for real-time process usage.
2. `free -h` for memory pressure.
3. `df -h` for disk capacity.
4. `iostat` and `vmstat` when troubleshooting bottlenecks.

Capture a baseline when your system is healthy so you can compare later.
