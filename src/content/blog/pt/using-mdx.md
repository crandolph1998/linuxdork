O MDX é útil quando um post em markdown comum precisa de um pequeno bloco de UI interativo ou reutilizável.

## Por que uso para notas de Linux

Mantenho a maioria dos posts em markdown puro, mas o MDX é ótimo para checklists repetidos.

<ul>
	<li>Confirmar que os backups estão saudáveis.</li>
	<li>Executar atualizações de pacotes com as notas de versão abertas.</li>
	<li>Reiniciar apenas após verificar os serviços em execução.</li>
</ul>

## Mantenha o MDX simples

Use MDX para pequenos fragmentos reutilizáveis, não para lógica de aplicativo inteira. Quanto mais limpos ficarem seus arquivos de conteúdo, mais fácil será mantê-los.
