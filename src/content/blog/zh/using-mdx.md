当普通 Markdown 文章需要一个小型的交互或可复用 UI 块时，MDX 就派上用场了。

## 为什么我用它来写 Linux 笔记

我的大多数文章都用纯 Markdown 编写，但 MDX 非常适合重复性的操作清单。

<ul>
	<li>确认备份状态正常。</li>
	<li>在打开 Release Notes 的情况下执行包更新。</li>
	<li>检查正在运行的服务后再重启。</li>
</ul>

## 让 MDX 保持简单

将 MDX 用于小型可复用片段，而非完整的应用逻辑。内容文件越整洁，维护就越轻松。
